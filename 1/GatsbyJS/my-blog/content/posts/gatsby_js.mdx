What is GatsbyJS?
- A static site generator
- Uses GraphQL to get data
- Uses React for templating
- Includes plugin architecture

Use GatsbyJS for speed, security and nice developer experience.

GatsbyJS is run inside NodeJS and using React for templating and CSS for styling

## Commands:


Create new gatsby project
> $ gatsby new "name" "link to theme is needed"

Start a develop instance of the gatsby project
> $ gatsby develop

Build for production
> $ gatsby build
